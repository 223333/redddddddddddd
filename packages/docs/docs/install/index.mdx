安装 Chronocat 只需要两步。

## 步骤 1：安装 QQ

在你的电脑上安装 QQ。

需要注意，并不是所有版本的 QQ 都支持下面的插件框架。如果你不知道应该选择哪个版本，我们推荐前往
WhichQQ 下载推荐的版本。
