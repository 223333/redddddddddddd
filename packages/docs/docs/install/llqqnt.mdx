## 步骤 1：安装 LiteLoaderQQNT

<UntrustedContents />

LiteLoaderQQNT 提供了详细的安装教程，可以直接前往 LiteLoaderQQNT 的主页查看。

<Link
  className="button button--secondary button--lg"
  to="https://llqqnt.mukapp.top"
>
  <span>前往 LiteLoaderQQNT</span>
</Link>

简单来说，从 [这里](https://github.com/LiteLoaderQQNT/LiteLoaderQQNT/releases) 下载最新版本的
LiteLoaderQQNT，解压得到 LiteLoader 文件夹后扔进
`C:\Program Files\Tencent\QQNT\resources\app` 目录，最后打开该目录下的 `package.json`
文件将 `main` 值修改为 `./LiteLoader` 即可，就像这样：

```diff
  {
   ...
   "homepage": "https://im.qq.com",
   "sideEffects": true,
-  "main": "./app_launcher/index.js",
+  "main": "./LiteLoader",
   ...
  }
```

结论：你已安装 LiteLoaderQQNT。

## 步骤 2：安装 Chronocat

启动 QQ 并登录机器人的账号后，在左下角的菜单中点击「设置」。

![「设置」](settings1.png)

你应当可以在设置中看到「LiteLoader 插件市场」。打开 LiteLoader 插件市场并找到
Chronocat，最后点击「安装」。

![LiteLoader 插件市场](llqqnt-market.png)

最后，打开「LiteLoader 配置界面」，然后点击「插件列表」下方的「扩展」，确保 Chronocat
已安装并已启动。

![确保 Chronocat 已安装](llqqnt-ensure.png)

结论：Chronocat 已成功安装在你的 QQ 上。
